# 💼 Omphile Hlongwane — Personal Portfolio Website

A personal portfolio website built with HTML, CSS, and JavaScript, showcasing my skills, projects and contact information.

---

## 👩🏽‍💻 About

This is the personal portfolio of **Omphile Hlongwane**, a Full Stack Developer & Data Analyst and BSc Application Development graduate from IIE Varsity College. The site highlights her background, technical skills, experience and featured projects.

---

## 🌐 Live Sections

| Section | Description |
|---|---|
| **Home** | Hero introduction with animated typing effect |
| **About** | Personal background, stats, and education details |
| **Skills** | Languages, frameworks, tools and concepts |
| **Projects** | Featured GitHub projects with descriptions |
| **Experience** | Professional work history as a timeline |
| **Contact** | Contact form and social media links |

---

## 🛠️ Built With

- **HTML5** — Semantic page structure
- **CSS3** — Custom dark theme with animations and responsive layout
- **JavaScript (Vanilla)** — Typing animation, scroll effects, hamburger menu, form handling
- **Google Fonts** — Inter & Space Grotesk
- **Font Awesome** — Icons
